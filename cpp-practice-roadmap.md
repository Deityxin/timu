# C++ 刷题路线图（入门 → 终极）

本路线按能力分层，覆盖数据结构、算法与竞赛常见专题。每层含建议题型与代表性题目（建议用 C++17，配合本仓库已有任务/脚本编译运行）。所有示例题统一使用洛谷（Luogu）题号与链接，便于在同一平台持续刷题。

— 练习建议 —
- 每日 3–6 题：1 道热身 + 2–4 道主线 + 1 道复盘/变式
- 同题型连续刷 20–40 题形成手感；每周做一次总结与错题回顾
- 模式：日常“正常模式”，模拟时切到“考试模式”（已配置快捷键）

— 环境 —
- 构建：VS Code 任务“C++: Build/Run Current File” 或手动 `g++ -std=gnu++17`
- 输入输出：建议使用 `bits/stdc++.h` 与 `ios::sync_with_stdio(false); cin.tie(nullptr);`

## 路线总览
- 入门基础：IO/语法/数组/字符串/模拟/简单数学
- 算法基础：排序、双指针、前缀和、差分、二分、哈希
- 数据结构：栈/队列/双端队列、堆、集合映射、并查集、树/图入门
- 递归与搜索：回溯、剪枝、状态设计
- 动态规划：线性 DP、区间 DP、背包、树形 DP
- 图论进阶：最短路、最小生成树、拓扑/SCC、二分图匹配
- 高级数据结构：Trie、线段树/树状数组、字典树应用、并查集进阶
- 算法进阶：LCA、倍增、重链剖分、莫队、启发式合并
- 数学与数论：素数筛、快速幂、逆元、同余、组合数、FFT/NTT
- 专家级：网络流（Dinic）、后缀结构（SA/SAM）、持久化、矩阵快速幂、点分治/树分治

> 说明：本路线统一以洛谷题号 P 开头为主（例如 P1048）。如果你更熟悉其他平台，可按主题自行替换为等价题。

## 在本仓库中如何快速开始
- 引入模板
  - 直接 `#include "d:/timu/templates/fast_pow.hpp"`（或相对路径 `"../templates/fast_pow.hpp"` 按你的文件布局调整）
  - 常用模板目录：`templates/`（DSU、Fenwick、SegTree、Dijkstra、KMP、LCA、fast_pow、sieve）
- 编译与运行（两种方式）
  - 任务：Ctrl+Shift+P → `Tasks: Run Task` → `C++: Run Current File`
  - 命令行（示例）：
    ```pwsh
    g++ -std=gnu++17 -O2 -Wall -Wextra -I d:\timu your_file.cpp -o your_file.exe
    .\your_file.exe
    ```
- 切换练习模式
  - 考试模式：`Ctrl+Shift+Space`（无联想，保留括号自动补全）
  - 正常模式：`Ctrl+Alt+1`（恢复联想与参数提示）
  - 切换后有系统弹窗提示，`.vscode` 目录下也会写 `MODE.exam` / `MODE.normal`
